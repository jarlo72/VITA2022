Deliverable 2: Interactive Web Map

The end product is a stand-alone interactive web-map application usable on Desktop (intended) or mobile browser with the following features:

  1. On-Screen or Mouse (see right image) Navigation controls: Provides user flexibility in viewing trail in desired perspectives (Zoom, Pan, Rotate, 3D perspective, Direction reset, home button, location tracking)
  2. Changeable Base maps: Flexibility on what surface user may overlay the trail data on
  3. Elevation Profile and Elevation Data: Useful metrics for planning such as slope, elevation gain, length of a selected or custom sketched profile on the map, in preferred units
  4. Layer Visibility Control: User controlled visibility Toggle. Zoom / Scale Dependant layering
  5. Dynamic Legend: Shows feature keys only to visible data within current window extent
  6. Viewable Attribute Data: Feature attributes and available spatial data upon click
  7. Layer Visibility Control: User flexibility in map visual
  8. Dynamic Legend: Shows feature keys only to visible data within the current window extent
  9. Viewable Attribute Data: Feature attributes and available spatial data upon clicking

Deliverable 3: Offline Map Data

The end product is a list of downloadable 30-inch by 30-inch Georeferenced PDFs Gase Maps（all 60 pdfs: PDF_Maps_ALL.zip ）and Feature Layer Data Files in KML/KMZ Format (all 12 external individual KMZ files: External_KMZ_ALL.zip ) including:

  1. One PDF base map in 1:500,000 scale: Download Scale_500k.pdf
  2. Nine PDF Base Maps in 1:100,000 Scale
    • Download sample maps Scale_100k_01.pdf, Scale_100k_02.pdf, and Scale_100k_03.pdf
    • Download all 9 maps: Scale_100k_ALL.zip
  3. Fifty PDF base maps in 1:20,000 scale
    • Download sample maps Scale_20k_01.pdf, Scale_20k_02.pdf, and Scale_20k_03.pdf
    • Download all 50 maps Scale_20k_ALL.zip
  4. Twelve (External) Feature Layer Data Files in KMZ Format (For Public Viewing and Download)
    • Download sample individual layers in KMZ Campsite.kmz, Limestone.kmz, Section 1.kmz, and Waterbody.kmz
    • Download a Navigation Package in KMZ VITA Navigation Package.kmz, which includes 8 layers (6 Trail Section layers, a Campsite layer, and a Trailhead & Parking layer), covering the entire Vancouver Island
    • Download an All External Layers Package in one KMZ External_Layer_ALL_20k_50.kmz for the 1:20,000 scale base map Scale_20k_50.pdf, which includes 12 layers (6 Trail Section layers, a Campsite layer, a Trailhead & Parking layer, a Watercourses layer, a Waterbody layer, a Limestone layer, and a Contour layer,), covering only the extent of the base map.
    • Download an All External Layers Package in one KMZ External_Layer_ALL_100k_09.kmz for the 1:100,000 scale base map Scale_100k_09.pdf, which includes 12 layers (6 Trail Section layers, a Campsite layer, a Trailhead & Parking layer, a Watercourses layer, a Waterbody layer, a Limestone layer, and a Contour layer,), covering only the extent of the base map.
  5. Four (Internal) Feature Layer Data Files in KMZ Format (not For Public Viewing and Download)
