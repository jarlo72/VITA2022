Early Planning - GEOM68 Project Planning

A general workflow was conceived during the project planning phase. The plan would be to achieve objectives and produce deliverables in logical sequence. From there, the workflows to develop the Interactive Web-Map and Offline Viewing Solution can be established concurrently to maximize time / scheduling.

The process illustrated in the flowchart above makes use of Web technologies, Cartographic Visualization Principles and Web Programming to bring the interactive web map to life.

Web Technologies

ArcGIS Online (AGOL) would be used as the primary Software-as-a-Service Product, being used as the cloud infrastructure to manage and consume the data layers into a 3-dimensional web-scene. ArcGIS Platform would be used to bring Esri mapping tools, data and location services into the web-map as interactivity features to enhance user experience and embed the web-scene onto a standalone Web Map application (see programming below). GitHub would be used as the hosting platform for this web map application.

Programming

Web-coding would use ArcGIS API for JavaScript to enable interactivity features and embed the web-scene as a stand-alone web map application onto an HTML webpage. HTML + CSS coding would be used for additional formatting that would aid in bolstering cartographic visualization of the user interface.
